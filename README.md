# Outdoor Recreation Asset Dashboard – Power BI Project

## 📊 Project Overview
This project is a **Power BI dashboard** created to analyze and visualize **Outdoor Recreation Assets**. The dashboard helps users understand asset distribution, categories, and usage trends through interactive visualizations.

The goal of this project is to demonstrate my skills in **data analysis, visualization, and business intelligence** using Power BI.

---

## 🧰 Tools & Technologies Used
- **Microsoft Power BI**
- **Data Cleaning & Modeling**
- **Data Visualization**
- **DAX (Basic Measures)**
- **GitHub (Project Documentation)**

---

## 📁 Dataset Description
The dataset includes information related to outdoor recreation assets, such as:
- Asset type and category  
- Location information  
- Asset count and distribution  
- Recreational usage data  

The data was cleaned and transformed in Power BI to ensure accuracy and consistency before creating visualizations.

---

## 📈 Dashboard Features
The Power BI dashboard includes:
- Interactive charts and graphs  
- Filters and slicers for better exploration  
- Clear visual representation of asset distribution  
- Insights into outdoor recreation patterns  

Users can easily explore the data and gain meaningful insights through the dashboard.

---

## 💡 Key Insights
- Identification of major outdoor recreation asset categories  
- Comparison of asset distribution across locations  
- Better understanding of recreation resource availability  

These insights can help in planning, resource management, and decision-making.
